# Contributing

Thanks for contributing to ZeroWay.

## Development
- Keep changes compatible with SDDM QML themes.
- Keep `theme.conf`, `README.md`, and runtime defaults in `Main.qml` aligned.
- Run `qmllint` before opening a pull request.

## Pull Requests
- Keep PRs focused and small.
- Include before/after screenshots or a short screen recording for UI changes.
- Mention any tested environments (distribution + desktop/session).

## Commit Style
- Use clear, imperative commit messages.
- Example: `Fix sidebar search default mismatch`.
